# 92系列环控器 Modbus-TCP 全量寄存器映射字典 (V2.0)

## 1. 系统控制与心跳区 (0x0000 - 0x0FFF)
| 地址 (Hex)    | 名称       | 类型      | 读写 | 换算倍率       |
| :------------ | :--------- | :-------- | :--- | :------------- |
| 0x0000        | 通信心跳值 | uint16    | R/W  | 原值           |
| 0x0001        | 控制模式   | uint16    | RO   | 0=自动, 1=手动 |
| 0x0002-0x000A | 设备ID     | uint8[18] | RO   | 字符串/HEX     |
| 0x000B-0x000D | MAC地址    | uint8[6]  | RO   | HEX数组        |
| 0x000E-0x000F | 固件版本   | uint32    | RO   | 原值           |
| 0x0010-0x0011 | 算法版本   | uint32    | RO   | 原值           |
| 0x0012-0x0013 | UI版本     | uint32    | RO   | 原值           |
| 0x0014-0x0015 | Boot版本   | uint32    | RO   | 原值           |
| 0x0016-0x0017 | 目标通风量 | uint32    | R/W  | 原值           |
| 0x0018        | 通风等级   | uint16    | R/W  | 原值           |

## 4. 外设硬件接口区 (0x2001 - 0x6000)

### 4.1 继电器与IO接口
| 地址范围      | 名称            | 说明                                                      |
| :------------ | :-------------- | :-------------------------------------------------------- |
| 0x2001-0x2016 | R1-R22 枚举     | 设备类型枚举 (1=风机组等)                                 |
| 0x2020-0x2029 | DI_1-DI_10 枚举 | DI绑定设备类型                                            |
| 0x2030-0x2033 | AO_0-AO_3 枚举  | AO绑定设备类型                                            |
| 0x4001        | 继电器状态      | uint32 位掩码 (bit0-21，对应R1-R22，**0-based，bit0=R1**) |
| 0x4003-0x4004 | AO1/AO2 反馈    | 风机转速反馈                                              |
| 0x4007        | DI输入状态      | uint16 位掩码 (bit0-9，**0-based，bit0=DI1**)             |
| 0x4008-0x400B | AI1-AI4 反馈    | 电压值 (val/10)                                           |
| 0x5001        | 继电器指令      | uint32 位掩码 (强制启停，**0-based，bit0=R1**)            |
| 0x5003-0x5004 | AO1/AO2 设定    | 风机转速设定                                              |

### 4.2 继电器位序对应关系 (关键：严禁偏移)
> R1 对应 bit0（最低位），R22 对应 bit21。
> relayIndex 传参规则：**0-based**，即 R1=0，R2=1，...，R22=21。
> value=true 表示置位该位（ON），false 表示清零该位（OFF）。
> 写入前必须先读取 0x5001 当前值，合并位图后再写回，严禁覆盖其他位。
