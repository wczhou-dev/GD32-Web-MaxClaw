# 通风逻辑 — 模块索引

## 架构层级

通风控制嵌入 `fan_control.c` 主线程，由三层逻辑协作：

| 层级 | 模块 | 职责 |
|------|------|------|
| 约束层 | [季节通风控制逻辑](季节通风控制逻辑.md) | 季节模式对等级的上下限约束 |
| 执行层 | [负压控制逻辑](负压控制逻辑.md) | 温度→等级计算、变频/定速/开口执行 |
| 风机模式层 | [间歇轮替关联间歇控制逻辑](间歇轮替关联间歇控制逻辑.md) | 风机间歇/轮替/关联间歇定时状态机 |

## 主入口

所有通风逻辑共享同一入口：
- 函数：`FanGroup_Control_Entry()`
- 线程创建：`fan_control_init()`
- 调度周期：20秒启动延时后持续运行

## 共享参数

以下参数在多个文档中引用，以 [负压控制逻辑](负压控制逻辑.md) 为权威定义方：

| 参数 | 说明 | 权威定义 |
|------|------|----------|
| `logicEnControl[]` | 逻辑使能表 | 负压控制逻辑 §2.1 |
| `SeasonType` | 季节模式类型 | 季节通风 §2.1 |
| `MinLevel` / `MaxLevel` | 等级上下限 | 负压控制逻辑 §2.1 |
| `_InterValCtr` / `s_interval_info` | 间歇状态机 | 间歇控制 §1.3 |

## 数据流

目标温度曲线 → `Expected_temp` → 温度差 `DeltaTemp` → 查表 → 等级 → 风机执行
                                        ↑
                              季节约束(MinLevel/MaxLevel)
