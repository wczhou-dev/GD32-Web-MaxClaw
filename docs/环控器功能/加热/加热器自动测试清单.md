## 1、测试概述

### 1.1 测试目标

本测试清单旨在全面验证环控器加热控制功能的正确性，包括：
- 室内/室外加热器的自动启停逻辑
- 加热器状态机转换的正确性
- 加热器与通风系统的联动逻辑
- 加热器与补偿系统的联动逻辑
- HMI参数设置与显示的正确性
- 异常场景下的保护机制

### 1.2 测试范围

| 测试类别 | 测试项数量 | 优先级 |
|:---|:---:|:---:|
| 室内加热器基础功能 | 12 | P0 |
| 室内加热器状态机 | 8 | P0 |

| 加热器长时间运行 | 4 | P2 |
| **总计** | **76** | - |

### 1.3 核心判定信号

| 判定维度 | 观测内容 | 合格口径 |
|:---|:---|:---|
| 继电器输出 | DO引脚电平 | 与加热器状态一致，状态变化时正确切换 |
| 运行状态 | `App_Run.IndoorHeating.State`、`App_Run.OutdoorHeating.State` | 与温度条件和参数配置一致 |
| 联动状态 | `App_Run.HeatRunState` | 状态转换正确：0→1→2→3→0 |
| 通风等级 | `App_Run.Ventilationlevel` | 加热期间等级不上升，加热解除后可正常调整 |
| 补偿逻辑 | 气体补偿、湿度补偿 | 加热期间不执行补偿，加热解除后恢复正常 |
| HMI显示 | 加热状态、参数回显 | 与内部运行态一致 |

## 3、室内加热器基础功能测试

### 3.1 室内加热器开启逻辑测试

#### 测试用例 HT-001：室内加热器正常开启（默认参数）

**测试目标**：验证室内加热器在温度低于开启阈值时能正确开启

**前置条件**：
- Expected_temp = 25℃
- indoor_openTemp = -2℃（开启阈值 = 25 + (-2) = 23℃）
- openchecktime = 0分钟（立即开启）
- IndoorHeating.State = 0（当前关闭）
- HeatRunState = 0（无加热状态）

**测试步骤**：
1. 设置室内温度为25℃（高于开启阈值）
2. 通过调试串口确认 `IndoorHeating.State = 0`
3. 缓慢降低室内温度至22℃（低于开启阈值23℃）
4. 记录温度变化到继电器输出的时间
5. 使用万用表测量室内加热继电器输出（应为PIN_HIGH）
6. 通过调试串口确认 `IndoorHeating.State = 1`
7. 通过调试串口确认 `HeatRunState = 1`

**预期结果**：
- 继电器在温度低于23℃后立即吸合（openchecktime=0时）
- `IndoorHeating.State` 从0变为1
- `HeatRunState` 从0变为1
- 继电器输出高电平

**自动化实现要点**：
```python
# 伪代码示例
def test_ht001():
    # 1. 设置初始条件
    set_expected_temp(25)
    set_indoor_open_temp(-2)
    set_open_check_time(0)
    set_indoor_heating_state(0)
    
    # 2. 设置温度高于阈值
    set_indoor_temp(25)
    wait(2)  # 等待系统稳定
    
    # 3. 降低温度至阈值以下
    set_indoor_temp(22)
    wait(1)  # 等待控制周期
    
    # 4. 验证继电器输出
    assert read_relay_output(INDOOR_HEATING_RELAY) == PIN_HIGH
    
    # 5. 验证内部状态
    assert read_indoor_heating_state() == 1
    assert read_heat_run_state() == 1
```

#### 测试用例 HT-004：室内加热器开启阈值验证（边界值）

**测试目标**：验证室内加热器在精确阈值边界的行为

- Expected_temp = 25℃
- indoor_openTemp = -2℃（开启阈值 = 23℃）
- openchecktime = 0分钟（立即开启）

**测试步骤**：

3. 验证继电器状态（应为关闭）
4. 设置室内温度为23.0℃（等于阈值）
5. 等待2个控制周期
6. 验证继电器状态（应为关闭，因为条件是 < 阈值）
7. 设置室内温度为22.9℃（低于阈值0.1℃）
8. 等待2个控制周期
9. 验证继电器状态（应为开启）

**预期结果**：
- 温度 >= 23℃时，加热器不开启
- 温度 < 23℃时，加热器开启

**自动化实现要点**：
```python
def test_ht004():
    # 测试边界值
    test_cases = [
        (23.1, False),  # 高于阈值，不开启
        (23.0, False),  # 等于阈值，不开启
        (22.9, True),   # 低于阈值，开启
    ]
    
    for temp, expected_state in test_cases:
        set_indoor_temp(temp)
        wait(2)  # 等待控制周期
        actual_state = read_relay_output(INDOOR_HEATING_RELAY) == PIN_HIGH
        assert actual_state == expected_state, f"温度{temp}℃时状态错误"
```
